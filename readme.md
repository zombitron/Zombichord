# Zombichord
Omnichord built with old phones

http://zombitron.org

## Installation
```
> git clone git@github.com:zombitron/Zombichord.git
> cd Zombichord
> npm install
> npm start
```

Dans le navigateur : 
- http://localhost:3000/ > interface cercle des quintes
- http://localhost:3000/strumplate > interface strumplate
